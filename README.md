# Statistics with R: Workflow and Exercises
This repository is based on 'Discovering Statistics with R' (Field, A., Miles, J., & Field, Z., 2012).
It serves as a documentation of my work based on the book for learning and performing statistical analysis in R. 
It covers a wide range of techniques, from foundational data visualization and assumption testing to complex multivariate modeling, following a structured workflow for reproducible research.


-----


# 📁 Repository Structure
The scripts are numbered to guide you through a typical statistical analysis progression:

### 1. Data Visualization & Exploration
`1_ggplot2.R:` Mastering the grammar of graphics for high-quality data visualization.

### 2. Foundational Checks & Basic Stats
`2_assumptions.R:` Testing for normality, homogeneity of variance, and other parametric requirements.

`3_correlation.R:` Bivariate, partial, and semi-partial correlations.

`4_regression.R:` Simple and multiple linear regression models.

`5_logistic_regression.R:` Binary and multinomial logistic regression for categorical outcomes.

### 3. Comparing Groups (T-Tests & ANOVA)
`6_t_test.R:` Independent-samples, paired-samples, and one-sample t-tests.

`7_ANOVA.R:` One-way Analysis of Variance.

`8_ANCOVA.R:` Analysis of Covariance (controlling for continuous covariates).

`9_factorial_ANOVA.R:` Multi-way ANOVA and interaction effects.

`10_repeatedmeasures_ANOVA.R:` Within-subjects designs.

`11_mixed_designs_ANOVA.R:` Combining between-subjects and within-subjects factors.

### 4. Advanced Multivariate Techniques
`12_nonparametric_practice.R:` Robust alternatives for non-normal data (e.g., Wilcoxon, Kruskal-Wallis).

`13_MANOVA.R:` Multivariate Analysis of Variance for multiple dependent variables.

`14_factor_analysis.R:` Exploratory Factor Analysis (EFA) and dimension reduction.

`15_categorical_data.R:` Chi-square tests and log-linear analysis.

`16_multilevel_linear_models.R:` Handling nested, hierarchical, or longitudinal data structures.
